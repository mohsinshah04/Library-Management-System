# Library-Management-System
A full-stack Library Management System built with Django and MySQL to streamline library operations. Students can search, reserve, and borrow books, while librarians manage users, inventory, loans, and fines. Features include role-based login, real-time database updates, overdue notifications, and a clean web GUI.  

To Run:  

Open Terminal 1 (Backend Server)  
cd DjangoProject  
.\.venv\Scripts\Activate.ps1  
python manage.py makemigrations  
python manage.py migrate  
python manage.py runserver   

Open Terminal 2 (Frontend Server)  
cd frontend  
npm start  
(click link)
